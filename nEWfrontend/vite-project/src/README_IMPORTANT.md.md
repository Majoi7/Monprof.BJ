# Informations importantes

- Tous les composants des pages sont dans `components`, et les pages dans `pages`.  
- Le CSS est directement dans le dossier `style` et aucun framework comme Tailwind n’a été utilisé.  
- Tout est lié via React Router, sauf le fichier `ResetPasswordPage`. Même si quelqu’un tente d’accéder directement à un fichier, les liens sont établis dans `app.jsx`.  
- Les fichiers ont été relus et documentés pour que chacun puisse s’y retrouver facilement.
